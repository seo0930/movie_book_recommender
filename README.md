# movie_book_recommender
영화/책 추천

1. 프로젝트 소개
이 프로젝트는 사용자의 '목적, 취향, 분위기 등'에 기반하여 영화와 책을 추천해주는 Streamlit 웹 애플리케이션입니다.
사용자는 간단한 체크박스와 선택지를 통해 자신의 기분·목적·취향을 입력합니다.
고른 선택지에 따라 각 결과(영화 또는 책)에 점수가 추가되고 가장 큰 점수를 가진, 즉 가장 어울리는 영화 또는 책을 추천해 줍니다.



2. 구현한 기능
 🎥  영화 추천 기능
- 선호 장르 선택
- 보고 싶은 분위기 선택
- 선호하는 영화 길이
- 위 요소를 조합해 가장 어울리는 영화 추천

 📖  책 추천 기능
- 읽는 목적 선택 (지식 / 감정 공감 / 위로 / 동기부여 / 재미)
- 선호 스타일 선택 (가벼움 / 묵직함 / 현실적 / 판타지 등)
- 위 요소를 조합해 추천 도서 결과 제공
